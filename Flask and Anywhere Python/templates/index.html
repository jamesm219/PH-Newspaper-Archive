PH Newspaper Archive

"This project daily scrapes the headlines of eight newspaper websites: Philippine Star, Tribune, Inquirer, Malaya, Manila Times, Manila Bulletin, Business World, and Rappler. It uses three python webscraping frameworks: BeautifulSoup, Scrapy, and Selenium to adapt to the news website's structure. For example, Selenium is used to cater for a website that mainly uses JavaScript like the Manila Bulletin. MySQL is used as primary Database Management System but the data is extracted to CSV for the project to be uploaded on PythonAnywhere using Flask for free. Twitter-roBERTa-base, a transformers model, from HuggingFace is used for Sentiment Analysis."
